# Restaurant Order and Billing Management System
This system helps a small restaurant manage their menu items, customer orders, and billing.
## Objectives
- Manage restaurant menu items.
- Accept customer orders.
- Calculate total bill amount.
- Apply discounts and taxes.
- Generate customer bills.

## Features
- Add menu items.
- Display menu.
- Search menu items.
- Remove menu items.
- Take customer orders.
- View ordered items.
- Calculate total order amount.
- Generate bills with tax and discount calculations.

## Installation
1. Download or clone the GitHub repository.
2. Open the project folder in VS Code or any Python IDE.
3. Ensure Python 3 is installed.
4. Run the program using:python main.py

## Project Files

- main.py – Starts the program
- menu_item.py – Contains the MenuItem class
- menu.py – Contains the Menu class
- order.py – Contains the Order class
- bill.py – Contains the Bill class
- restaurant.py – Controls the restaurant system

- ## Classes Used

#### Menu_Item
Represents a single menu item.
#### Menu
Manages all menu items.
#### Order
Manages customer orders.
#### Bill
Calculates and generates bills.
#### Restaurant
Controls the overall restaurant system.
